# Contract: i18n (`src/i18n/`)

Pure localization core. No DOM, no storage, no globals. Implements FR-001..FR-006, FR-005
fallback, and SC-001.

## Module surface

```ts
// src/i18n/index.ts
export type Lang = 'en' | 'uk';
export const LANGS: readonly Lang[];
export const DEFAULT_LANG: Lang;            // 'en'

export type Catalog = Record<string, string>;
export const messages: Record<Lang, Catalog>;

export function t(lang: Lang, key: string, vars?: Record<string, string>): string;
export function detectLang(navigatorLanguage: string): Lang;
```

`en.ts` and `uk.ts` each `export default` (or named) a `Catalog`.

## `t(lang, key, vars?)`

- **Resolution order**: `messages[lang][key]` → `messages.en[key]` → `key`.
- **MUST NOT** return an empty string when `messages.en[key]` exists.
- **Interpolation**: replace each `{name}` token in the resolved string with `vars.name`.
  Tokens with no matching var are left as-is. No throwing.
- **Purity**: same inputs → same output; no I/O.

### Behavior table

| lang | key in uk? | key in en? | vars | result |
|------|-----------|-----------|------|--------|
| uk | yes | yes | — | uk string |
| uk | no | yes | — | en string (fallback, FR-005) |
| en | — | yes | `{name:'V'}` on `"Made by {name}."` | `"Made by V."` |
| uk | no | no | — | the key itself (last-resort; forbidden for real keys by parity test) |

## `detectLang(navigatorLanguage)`

- Lowercase the input; if it `startsWith('uk')` or `startsWith('ru')` → `'uk'`; else `'en'`.
- Pure; `''`/garbage → `'en'`.

| input | output |
|-------|--------|
| `'uk'`, `'uk-UA'`, `'UK'` | `uk` |
| `'ru'`, `'ru-RU'` | `uk` |
| `'en-US'`, `'de'`, `''`, `'zz'` | `en` |

## Catalog parity (test-enforced — SC-001 / I-L2)

- Every key present in `uk` MUST also exist in `en`.
- For every key the UI renders, `messages.en[key]` MUST be a non-empty string.
- A unit test iterates `Object.keys(messages.en)` and asserts `t('uk', key)` is never blank
  and never equal to the raw key.
